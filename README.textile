== MyBudGet

Using this APP you are able to control your finances. You can add new Expense categories, Income Categories, Income Tranactions, Expense Transactions. In a simple way you can plan a category by adding the future expenses. You can see detailed statistics of incomes and expenses, divided by category, days, and other criteria.
